# Community Repertoire Indexes

This repository contains **JSON manifests** and **metadata indexes** created by the community to help musicians organize their libraries.

## ⚠️ Disclaimer ⚠️

1.  **No Files Hosted:** This repository **does not host, distribute, or store** any PDF files, sheet music, or copyrighted audio material.
2.  **Metadata Only:** The files herein are strictly JSON text files containing metadata (titles, composers, page numbers) and pointers to external resources found publicly on the internet.
3.  **Third-Party Content:** Any links to external content are provided by community contributors. The maintainers of this repository have no control over external websites.
4.  **Copyright:** Users are responsible for ensuring they have the legal right to access any content linked in these manifests. If you own a physical copy of a book, you may be entitled to a digital backup copy depending on your local laws.

## How to use

Copy the "Raw" link of any `.json` file and paste it into your compatible repertoire manager app.
